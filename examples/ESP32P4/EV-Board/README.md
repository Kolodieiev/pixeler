Приклад прошивки для плати ESP32-P4-Function-EV-Board.

Різні покоління цієї плати можуть мати відмінності, наприклад, у версії чипа ESP32P4.
Через такі відмінності можливо буде необхідно змінити джерело тактування в graphics_config.h для коректної роботи дисплею, тощо.

Керування прошивкою виконується дотиками на екрані, свайпами вліво/впарво/вгору/вниз, а також довгим утримуванням на віджетах.
