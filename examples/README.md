ВАЖЛИВО!!! Команди для збірки та встановлення прикладів прошивок гарантовано працюватимуть тільки в ОС Linux.
На інших ОС необхідно самостійно налаштувати процес коректної збірки.


В цьому каталозі знаходяться приклади прошивок для різних плат на ESP32/S3/P4.
В більшості випадків приклади прошивок не є функціонально завершеними, 
та покликані лише продемонструвати коректні налаштування фреймворку для плати.
Окрім того можуть містити копіпастний код з оригінальних прошивок, 
який не використовується для певної плати через особливості її заліза.


Підкаталог `scripts` є службовим. Його не потрібно модифікувати чи видаляти.


Для завантаження на плату зібраного двійкового файлу, перейдіть до каталогу плати.
Відкрийте термінал в папкі `bin`.
Виконайте команди з файлу README.


Для редагування та самостійної збірки прошивки, відкрийте кореневий каталог плати в VS Code.
Усі приклади прошивок збираються плагіном Pioarduino та не можуть бути зібрані за допомогою Platformio!
